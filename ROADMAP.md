# Quick Anime Renamer Redux Roadmap

This document tracks the current release and the next areas of work for **Quick Anime Renamer Redux**.

The project prioritizes:
- Predictable behavior
- Offline, local-only operation
- Minimal UI complexity
- Opt-in automation

---

## Current Stable Version

**v1.2.0**

Released features:
- Stable persistence of settings
- Portable INI-based configuration
- Reliable startup and shutdown behavior
- Two-column rename preview
- Undo rename support
- Auto-load last directory on startup
- Delete key removes selected rows from the batch
- Filename conflict detection before rename
- Remember window size and position
- Remove standalone version tags like `v2` and `v3`

---

## Future Direction

Future releases may explore:
- Recursive subfolder scanning
- Additional filename cleanup rules
- More conflict-resolution options
- UI refinements based on user feedback
- Optional copy-to-clipboard preview export

All future work will continue to prioritize simplicity and user control.
